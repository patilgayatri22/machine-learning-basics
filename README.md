# 🧠 Machine Learning: Complete Crash Course

> A structured, hands-on guide to Machine Learning — from raw data to production systems.  
> Designed for developers, data scientists, and curious learners.

---

## 📚 Course Index

| # | Module | Topics | Time |
|---|--------|--------|------|
| 01 | [ML Data: Numerical & Categorical](./01_ML_Data_Numerical_Categorical.md) | Data types, encoding, scaling, pipelines | 45–60 min |
| 02 | [Datasets, Generalization & Overfitting](./02_Datasets_Generalization_Overfitting.md) | Train/val/test splits, bias-variance, regularization, CV | 50–65 min |
| 03 | [Advanced ML Models](./03_Advanced_ML_Models.md) | Random Forest, XGBoost, LightGBM, SVM, Neural Nets | 90–120 min |
| 04 | [Production ML Systems](./04_Real_World_ML_Production.md) | MLOps, APIs, monitoring, drift detection | 75–90 min |
| 05 | [AutoML](./05_AutoML.md) | FLAML, AutoGluon, TPOT, Optuna, NAS | 60–75 min |
| 06 | [ML Fairness](./06_ML_Fairness.md) | Bias, fairness metrics, mitigation, model cards | 60–75 min |

**Total:** ~6–8 hours of study + exercises
